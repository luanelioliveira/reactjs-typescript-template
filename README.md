## ⚡️ Getting started

1. Clone it

   ```sh
   git clone git@github.com:luanelioliveira/reactjs-typescript-template.git
   ```

2. Install dependencies

   ```sh
   cd reactjs-typescript-template
   yarn
   ```
2. Start app

   ```sh
   yarn start
   ```

3. Are you ready for launch?

   Your site is now running at `http://localhost:3333`
